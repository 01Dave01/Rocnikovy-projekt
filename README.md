# Ročníkový projekt
David Kaněra

T3A

## Popis projektu:
Můj ročníkový projekt je na informačního displeje do počítače. Displej slouží k zobrazování informací jako je aktuální čas a teplota. Mým cílem je mít přehled o teplotách počítače při různých hrách bez otevírání jiného programu. Displej je připojen k mikrokontroleru Pico 2 W, který zpracovává data. Bohužel stále čekám než mi přijde displej, abych mohl začít něco pořádně dělat a mít toho více.

## Cíl projektu:
- vytvořit funkční extérní displej k počítači
- zobrazovat aktuální čas a teplotu
